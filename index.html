Графика и построение изображений на языке C++

На C ++ программирование графического пользовательского интерфейса (GUI) важно при разработке современных приложений, где у пользователей есть приятная графика для работы. Хотя C ++ обычно связан с системным программированием и написанием игр, он может быть отличной альтернативой написанию GUI.

Рассмотрим несколько графических библиотек, которые позволяют сделать разработку UI приложений максимально удобной

  • Библиотека QT

    QCustomPlot — это виджет Qt C++ для построения графиков и визуали-зации данных. Он не имеет дополнительных зависимостей и хорошо докумен-тирован. Эта библиотека построения графиков предназначена для создания красивых 2D-графиков, графиков и диаграмм издательского качества Устанавливается QtCreator, загружается виджет QCustomPlot и добавляются его файлы в проект. После чего данный виджет будет готов к использованию, достаточно всего лишь перенести его на редактор графических форм.

    Код программы

    QVector<double> x(101), y(101); for (int i=0; i<101; ++i) { x[i] = i/50.0 - 1; // x идёт от -1 до 1 y[i] = x[i]*x[i]; // зададим квадратичную функцию } // создаём график и назначаем данные: customPlot->addGraph(); customPlot->graph(0)->setData(x, y); // подпишем оси: customPlot->xAxis->setLabel("x"); customPlot->yAxis->setLabel("y"); // установим диапазоны осей, чтобы были видны все данные: customPlot->xAxis->setRange(-1, 1); customPlot->yAxis->setRange(0, 1); customPlot->replot();

  • Библиотека graphics.h

    Библиотека Graphics в C++ предназначена для создания графической модели, например, для создания различных фигур и добавления к ним цветов. Это можно сделать в консоли C++, импортировав библиотеку graphics.h в компилятор GCC. Мы также можем нарисовать круг, линию и другие геометрические фигуры. Формальный синтаксис приведен в следующем виде:

    # include<graphics.h> { Initgraph(); }

    Код программы

    #include <graphics.h> #include <iostream> int main() { int gd = DETECT, gm; initgraph(&gd, &gm, ""); line(140, 140, 350, 100); line(140, 140, 200, 200); line(350, 140, 200, 200); getch(); closegraph(); }
